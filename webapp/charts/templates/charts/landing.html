Global Spotify Hits – 2023 Charts Overview

This dashboard explores a 2023 Spotify charts dataset, focusing on three questions:

  • Which songs are the biggest global hits by total streams?
  • Which songs appear in the largest number of countries' charts?
  • How diverse are different countries' charts in terms of unique tracks?

Top Global Streams

Ranks songs by total global streams, combining counts across all included countries for 2023.

Demonstrates aggregation using the Django ORM and plotting with Matplotlib.

View Top Streams

Global Hit Songs

Shows which songs appear in the largest number of country charts, giving a different view of what it means to be a "global hit."

Uses distinct counts over countries and highlights songs that travel well across regions.

View Global Hits

Country Diversity

Compares how many unique tracks appear in each country's chart, which reflects how varied different markets are.

Uses group-by logic to count distinct tracks per country and plots the results in a bar chart.

View Diversity

Database Browser

Search the SQLite-backed dataset by song, artist, country, month, and explicit flag. Useful for digging into specific songs or regions.

Demonstrates server-side filtering, pagination, and a simple table view over the raw chart rows.
